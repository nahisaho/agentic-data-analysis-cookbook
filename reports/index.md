# AI エージェント時代のデータ分析入門

AI Agent と MCP（Model Context Protocol）を用いて、実験データの分析を自然言語で行うための実践書です。

- **対象読者**: ARIMデータポータル会員（材料・ナノテク研究者。Python/Jupyter経験あり、MCP未経験）
- **最終ゴール（合格ライン）**: 読者が自分の実験データ向けに「データ分析用Skill」を1つ以上、自力で作成できること
- **標準環境**: Python + JupyterLab + Jupyter MCP + GitHub Copilot CLI + ToolUniverse MCP ほか（FastMCP, arXiv MCP, Paper Search MCP）

> [!NOTE]
> 本書は多様な装置カテゴリを **6つのデータ型**（スペクトル型／クロマトグラム・時系列型／画像・顕微鏡型／回折・散乱パターン型／表形式・プロセス条件型／マルチモーダル統合型）に抽象化して扱います。1つの型でSkillを作れれば、同じ型の他装置へ骨格を転用できます。
